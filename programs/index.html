Development

I use Neovim for almost everything. I find debugging much more convenient in standard IDEs, so I use VSCodium, which is truly open-source VS Code.

When in need for tabs I go with tmux.

I use Hugo for making websites. It’s blazing fast and very easy to use. The community is very active, so if I run into any issues I can easily find the answer online.
